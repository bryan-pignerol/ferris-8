## About
Ferris-8 is what we called a "Fantasy Console". You can code your game in Lua language.

> #### Warning
> This project is part of my learning process for the Rust programming language and is not as fully developed as existing alternatives. If you're looking for a fantasy console to develop your games, I recommend checking out other software such as PICO-8 or TIC-80.
